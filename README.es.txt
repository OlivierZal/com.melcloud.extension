Esta aplicación extiende la funcionalidad de la aplicación Homey MELCloud ajustando automáticamente la temperatura de enfriamiento de la bomba de calor aire-aire con una desviación máxima de 8°C de la temperatura exterior.

¿Por qué?

- Porque es mejor para sus bombas de calor.
- Porque es mejor para su salud.
- Porque es mejor para el medio ambiente.

Requisitos previos:

- Debe tener un Homey Pro.
- La aplicación Homey MELCloud debe estar instalada.
- Se debe instalar una aplicación Homey que devuelva la temperatura exterior.
