Questa app estende l'app MELCloud per Homey regolando automaticamente i tuoi dispositivi aria-aria per mantenere una temperatura di raffrescamento entro 8 °C dalla temperatura esterna.

Perché?

- Perché è meglio per le tue pompe di calore.
- Perché è meglio per la tua salute.
- Perché è meglio per l'ambiente.

Prerequisiti:

- Devi avere un Homey Pro.
- L'app MELCloud per Homey deve essere installata.
