Это приложение расширяет приложение MELCloud для Homey: устройства воздух-воздух автоматически настраиваются так, чтобы температура охлаждения оставалась в пределах 8 °C от наружной температуры.

Почему?

- Потому что это лучше для ваших тепловых насосов.
- Потому что это лучше для вашего здоровья.
- Потому что это лучше для окружающей среды.

Требования:

- Необходим Homey Pro.
- Должно быть установлено приложение MELCloud для Homey.
